OVERVIEW

RESTFUL API Doctor Landpage that validates user submission input with FrontEnd endpoints that communicates with the Backend Server for redirection to Weclome page. 
Typescript, express, node js & react


FEATURES:

  -RESTFUL API endpint to routes using common methods like 
GET (to get data),
 POST (to send data),
 PUT (to update data), and 
DELETE (to remove data).

- json web token for secure authentication
- User credentials (username, password, and email) are stored in memory
-   Securely handles the token-based authentication
-   set a cookie in HTTP response
-   Express Cors, Helmet, Express-Rate-Limit, Json-web-token- , etc modules
